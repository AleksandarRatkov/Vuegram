# Vuegram
A simple social media web app built with Vue.js, Vuetify and Firebase's Cloud Firestore

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
